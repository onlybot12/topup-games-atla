Bantuan & FAQ

Cara Top Up Game?
1. Pilih game yang ingin di top up
2. Masukkan User ID (dan Zone ID untuk Mobile Legends)
3. Pilih nominal yang diinginkan
4. Scan QRIS yang muncul
5. Tunggu 1-5 detik, diamond/item akan masuk otomatis
Cara Beli Akun Premium?
Pilih layanan Premium (Spotify/YouTube/dll), masukkan email aktif Anda. Data akun akan dikirim via SN atau email setelah pembayaran berhasil.
Transaksi Gagal/Pending?
Jika saldo terpotong namun status gagal, hubungi admin dengan screenshot bukti transfer dan TRX ID (RAYxxxxx).
Bagaimana cara cari transaksi saya?
Klik menu "Cari Transaksi" di sidebar, lalu masukkan Order ID atau Deposit ID Anda. Transaksi akan muncul jika ditemukan dalam sistem.
Berapa lama proses pengiriman?
Setelah pembayaran terverifikasi, item akan diproses otomatis dalam 1-5 detik. Untuk beberapa produk tertentu mungkin memerlukan waktu hingga 5 menit.
